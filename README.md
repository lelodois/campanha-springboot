# Campanha 

É uma api REST que disponibiliza serviços de campanha, usuario e associação dos mesmos.

## Arquitetura

* RestAPI
* Java 8
* Maven
* Spring boot / Mvc / Event
* Tomcat embedded
* Hibernate / Listener
* Swagger
* H2
* Junit
* Mockito

## Getting Started

Run CampanhaApplication.java
http://localhost:9999/swagger-ui.html
